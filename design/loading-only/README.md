# 퀴즈 생성 중 (로딩 화면)

모델이 실시간으로 문제를 생성하는 동안 보여주는 로딩 화면입니다.
스크린샷: `09-loading.png` (모바일 390×844, 라이트)

## 구성
- **회전하는 몬스터볼** + 점멸하는 점 3개
- 제목 "퀴즈 만드는 중" + 설명 문구
- 좌우로 흐르는 **불확정(indeterminate) 진행바**
- **단계 체크리스트**
  1. 메타 포켓몬 불러오는 중 — ✓ 완료 (초록)
  2. 스피드 공식 계산 중 — ✓ 완료 (초록)
  3. 문제 만드는 중 — 회전 스피너 (진행 중, 파랑)
  4. 정답 · 해설 준비 중 — 빈 원 (대기)
- 하단 안내: "잠시만 기다려 주세요 · 보통 몇 초면 끝나요"

## 동작
- `/quiz/generate` 호출 동안 이 화면을 표시하고, 응답이 오면 퀴즈 화면으로 전환.
- 단계 체크리스트는 실제 진행 상황에 맞춰 ✓/스피너/대기 상태를 갱신하면 됨.

## 토큰 (HeroUI v3 기준)
- Primary `#006FEE` (스피너·진행바·점), Success `#17C964` (완료 ✓)
- 배경 `#fafafa` + 상단 방사형 `#E6F1FE`, 폰트 Pretendard
